클래스의 덩치가 커진다는 것은 해당 객체의 책임도 많아 진다는 뜻입니다.
따라서 단 한가지 이유만으로 클래스를 수정 할 수 있어야 하며, 여러가지 이유로 수정할 이슈가 생긴다면 그건 설계가 잘못된 것이라고 합니다.
